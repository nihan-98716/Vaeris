This directory holds only the `metadata.json` for the model version that
produced `docs/ablation_results.md` — proof of a real training run, not a
placeholder. The trained model binaries (`model_q10.txt`, `model_q50.txt`,
`model_q90.txt`) are intentionally NOT included here, matching the ML Model
Specification (Section 6.10): only metadata is meant to be committed to git;
binaries belong in `.gitignore` and get regenerated by re-running
`quantile_lgbm.py` locally (see README.md at the project root).
