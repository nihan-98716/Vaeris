# Vaeris Data Directory

This directory stores datasets, historical snapshots, and database resources for the Vaeris platform.

## Subdirectories:
* `raw/` - Raw historical sensor/weather datasets (gitignored except for this README).
* `processed/` - Processed outputs and clean CSVs (gitignored except for this README).
* `snapshots/` - Static JSON files representing baseline demo scenarios (e.g., Delhi Nov 13-18, 2024 event).
